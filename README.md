# Exercícios em C #

Este repositório contém programas simples em linguagem C para praticar conceitos básicos como loops, arrays, strings e lógica.

######______####### 

atv1. Impressão de Números de 1 a N: 
Imprime todos os números de 1 até N informado pelo usuário.

atv2. Cálculo de Fatorial
Calcula o fatorial de um número inteiro.

atv3. Soma de Números em um Array
Lê 5 números inteiros, armazena em um array e calcula a soma.

atv4. Conversão de Temperatura  
Converte temperaturas entre Celsius e Fahrenheit.

atv5. Contagem de Caracteres em uma String  
Conta quantos caracteres existem em uma string.

atv6. Verificação de Número Primo
Verifica se um número é primo.
